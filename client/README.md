# wety-client

The frontend for [`wety.org`](https://www.wety.org/). 

It's a [Vite](https://vite.dev/) app using [SolidJS](https://www.solidjs.com/) with [D3](https://d3js.org/) for tree rendering and [@ark-ui/solid](https://ark-ui.com/) for UI components.

## Local development

You'll need the `wety` API server running first — see the root README for that.

```bash
npm install
npm run dev
```

The dev server starts at `http://127.0.0.1:8000`. Changes are hot-reloaded.
